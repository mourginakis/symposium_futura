# Symposium Futura

### A community tech blog hosted on the Internet Computer (IC) Blockchain.


Symposium Futura is a community-owned blog that aims to incentivize and curate high-quaility community-written information about the future of tech.

The site will be secured using Internet Identity (II), and read only for anonymous users. Any logged-in user can become an author, and create a blog post. Authors can gain recognition and status, through the token system. New posts can also be incentivized with our tokenomics system.
